CloudVigia - Projeto Final
Sobre o Projeto
CloudVigia é uma plataforma web desenvolvida como projeto final para conclusão do curso do programa re/Start e Inteligência Artificial na Escola da Nuvem.
O sistema simula uma plataforma de monitoramento urbano inteligente que utiliza IA para combater o descarte irregular de resíduos e prevenir alagamentos urbanos através da participação comunitária.

Objetivo do Trabalho
Demonstrar a aplicação prática dos conhecimentos adquiridos no curso, integrando:
Desenvolvimento Web (HTML, CSS, JavaScript)
Arquitetura em Nuvem
Interface de Usuário
Manipulação de Dados
Sistemas Inteligentes

Como Executar
Baixe os arquivos do projeto
Abra o arquivo index.html em um navegador moderno
Explore todas as funcionalidades

Tecnologias Utilizadas
Frontend: HTML5, CSS3, JavaScript (ES6+)
Bibliotecas: Leaflet.js (mapas), Chart.js (gráficos), Font Awesome
Responsivo: Mobile-first approach

Funcionalidades
Principais
Mapa interativo com denúncias
Formulário de denúncia inteligente
Dashboard com estatísticas
Gráficos de análise de dados
Sistema de acompanhamento

Técnicas
Upload de imagens (drag & drop)
Geolocalização automática
Validação de formulários
Notificações em tempo real
Dados fictícios realistas
Design: CSS Grid, Flexbox, Variáveis CSS
